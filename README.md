# minha-opp

PWA de onboarding gamificado para colaboradores do Grupo Opp+.

Guia interativo mobile-first que conduz o colaborador na configuracao do email corporativo (@grupooppmais.com.br) no Gmail pessoal.

## Stack

- HTML + CSS + JavaScript (vanilla)
- Design System Grupo Opp+ v1.1
- Hospedagem: Vercel (free tier)

## Desenvolvimento

```bash
# Servir localmente
npx serve public/
```

## Estrutura

```
public/           # Arquivos servidos (PWA)
  css/            # Estilos e design tokens
  js/             # Logica da aplicacao
  assets/         # Imagens e icones
  index.html      # Entrada principal
docs/             # Documentacao do projeto
```
